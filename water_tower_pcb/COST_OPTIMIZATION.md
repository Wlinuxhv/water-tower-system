## 🛠️ 固件适配

### STC8G1K08 固件框架

```c
// 精简版固件框架 (目标 < 7KB)
#include <stc8g1k08.h>

// 定义 (使用位域节省 RAM)
typedef struct {
    uint8_t address : 2;    // 2 位地址
    uint8_t mode : 1;       // 自动/手动
    uint8_t pump : 1;       // 水泵状态
    uint8_t alarm : 1;      // 报警标志
    uint8_t reserved : 3;
} SystemFlags;

SystemFlags flags = {0};

// 精简的 LoRa 通信协议
#define CMD_HEARTBEAT  0x01
#define CMD_PUMP_ON    0x10
#define CMD_PUMP_OFF   0x11
#define CMD_QUERY      0x02

// 主循环 (精简版)
void main(void) {
    System_Init();
    
    while(1) {
        if (LoRa_Received()) {
            Process_Command();
        }
        
        if (Timer_Expired()) {
            Send_Heartbeat();
        }
        
        Read_ADC();
        Check_Water();
        Control_Pump();
    }
}
```

### 内存映射

| 区域 | 地址 | 大小 | 用途 |
|------|------|------|------|
| 代码区 | 0x0000-0x1FFF | 8KB | 程序代码 |
| RAM | 0x30-0x7F | 80B | 直接寻址 RAM |
| XRAM | 0x0000-0x03FF | 1KB | 扩展 RAM |
| EEPROM | 0x2000-0x27FF | 2KB | 数据存储 |

## ⚠️ 注意事项

### STC8G1K08 限制

1. **Flash 仅 8KB**: 需精简程序，避免大功能
2. **RAM 仅 1KB**: 减少全局变量，复用缓冲区
3. **封装 SOP16**: 引脚有限，需合理复用
4. **ADC 10 位**: 精度略低于 STC32G(12 位)，但够用

### 生产注意事项

1. **SMT 贴片**: STC8G1K08 SOP16 需注意方向
2. **焊接温度**: SOP16 封装，烙铁温度 350°C
3. **程序烧录**: 使用 STC-ISP，选择正确型号
4. **测试**: 先测试基本功能，再测试通信
