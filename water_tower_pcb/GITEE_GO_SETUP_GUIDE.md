# 📖 Gitee Go 服务开通指南

## ⚠️ 为什么无法启动？

Gitee Go 是 Gitee 的付费/免费增值服服务，需要**先开通**才能使用。

---

## ✅ 开通步骤（手机操作）

### 方法 1：通过仓库页面开通

1. **打开 Gitee App** 或访问网页
   - 网址：https://gitee.com/wlinuxhvxx/water-tower-task

2. **进入服务页面**
   - 点击顶部导航栏的 **"服务"** 或 **"Gitee Go"**
   - 或者点击 **"设置"** → **"服务管理"**

3. **找到 Gitee Go**
   - 在服务列表中找到 **"Gitee Go"** 或 **"持续集成"**
   - 点击 **"开通"** 或 **"启用"**

4. **选择套餐**
   - **免费套餐**：公共仓库 2000 分钟/月，私有仓库 500 分钟/月
   - 建议选择免费套餐先试用

5. **确认开通**
   - 阅读服务条款
   - 点击 **"确认开通"**

---

### 方法 2：通过 Gitee Go 主页开通

1. 访问：https://gitee.com/features/gitee-go
2. 点击 **"立即使用"** 或 **"免费开通"**
3. 登录您的账号（已登录）
4. 选择要开通的仓库：`water-tower-task`
5. 确认开通

---

### 方法 3：联系客服开通

如果以上方法都不行：

1. 访问：https://gitee.com/enterprises
2. 点击 **"在线咨询"** 或拨打 **400-606-0201**
3. 说明要开通 Gitee Go 服务
4. 客服会帮您开通

---

## 🔍 开通后的操作

开通成功后：

1. 回到仓库页面：https://gitee.com/wlinuxhvxx/water-tower-task
2. 点击 **"服务"** → **"Gitee Go"**
3. 应该能看到 `.gitee/go.yml` 配置文件
4. 点击 **"运行"** 或 **"手动触发"** 开始构建

## 🎯 替代方案

如果 Gitee Go 开通困难，可以考虑：

### 方案 A：使用 GitHub Actions（推荐）
1. 在 GitHub 创建账号
2. 创建仓库并推送代码
3. GitHub Actions 自动免费构建（2000 分钟/月）
4. 下载 APK

### 方案 B：本地构建（需要电脑）
1. 安装 Android Studio
2. 打开项目
3. 点击 Build → Build APK
4. 生成 APK 文件

### 方案 C：使用其他 CI 服务
- Travis CI
- CircleCI
- Jenkins
