
### 3️⃣ 演示水泵控制
1. 点击任意水塔卡片
2. 弹出控制对话框
3. 点击"开启/停止水泵"
4. 展示状态变化

### 4️⃣ 展示历史记录
1. 点击右上角菜单 → **历史**
2. 点击"选择日期"
3. 展示水位曲线图
4. 指出最高/最低/平均水位

### 5️⃣ 说明真实部署效果
- 实际连接 ESP8266 后数据来自真实传感器
- 演示模式用于展示界面和交互流程
- 真实部署时关闭演示模式即可

## ⚙️ 切换模式

### 演示模式 → 真实模式
1. 点击右上角 **信息图标** ℹ️
2. 提示"已切换到真实模式"
3. 进入设置配置 ESP8266 IP 地址
4. 测试连接

### 真实模式 → 演示模式
- 连接失败时自动切换
- 或手动点击信息图标切换
