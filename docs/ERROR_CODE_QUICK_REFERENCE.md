# 错误码速查卡

## 快速诊断表

### 🚨 严重错误 (立即停机)

| 错误码 | 显示 | 含义 | 应急处理 |
|--------|------|------|----------|
| **E004** | Power Low | 电源电压低 | 更换电源适配器 |
| **E101** | Well Water Low | 井水缺水 | 等待水源恢复 |
| **E301** | Relay Stuck On | 继电器粘连 | 断开电源，更换继电器 |
| **E303** | Relay Overload | 继电器过载 | 断开大功率水泵 |
| **E501** | Tower Overflow | 水塔溢出 | 关闭进水，检查传感器 |
| **E503** | Pump Fail | 水泵故障 | 检修水泵电机 |

### ⚠️ 警告错误 (注意观察)

| 错误码 | 显示 | 含义 | 建议处理 |
|--------|------|------|----------|
| **E002** | Mem Low | 内存不足 | 重启系统 |
| **E103** | Sensor Range | 数据超范围 | 校准传感器 |
| **E205** | WiFi Fail | WiFi 断开 | 检查路由器 |
| **E401** | No Slave Data | 无从机数据 | 检查从机供电 |
| **E502** | Tower Dry | 水塔干涸 | 检查进水阀门 |

## 错误处理流程图

```
开始
  │
  ▼
OLED 显示错误码
  │
  ▼
记录错误码 (Exxx)
  │
  ▼
查表确定错误类型
  │
  ├── 严重错误 (CRIT) ──→ 立即停机 ──→ 联系维修
  │
  ├── 错误 (ERROR) ──→ 尝试恢复 ──→ 失败则报修
  │
  ├── 警告 (WARN) ──→ 继续观察 ──→ 恶化则处理
  │
  └── 信息 (INFO) ──→ 记录日志 ──→ 无需处理
```

## 一键诊断命令

### Web 界面诊断

访问：`http://主机 IP/api/status`

返回示例:
```json
{
  "wifi": true,
  "mode": "AUTO",
  "well_water": false,
  "towers": 3,
  "error_code": 101,
  "error_level": "CRITICAL"
}
```

### 串口诊断

发送：`STATUS`  
返回：`System Status: ERROR E101 - Well Water Low`

### 清除错误

Web 界面：`http://主机 IP/api/error/clear`  
串口命令：`ERROR CLEAR`

## 应急处理预案

### 预案 1: 井水缺水 (E101)

1. **自动响应**: 系统自动关闭所有水泵
2. **人工确认**: 检查水源是否真的缺水
3. **恢复供水**: 等待水源恢复
4. **系统重启**: 自动恢复运行

### 预案 2: 继电器粘连 (E301)

1. **自动响应**: 系统报警，尝试断开
2. **紧急断电**: 立即断开 220V 电源
3. **更换继电器**: 更换故障继电器
4. **测试运行**: 确认正常后恢复供电

### 预案 3: 水塔溢出 (E501)

1. **自动响应**: 关闭水泵，打开排水阀 (如有)
2. **人工干预**: 关闭进水阀门
3. **检查原因**: 传感器故障 or 控制失效
4. **维修恢复**: 更换传感器或检修电路
