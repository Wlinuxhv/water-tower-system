## 主机 (ESP8266) 详细电路

### 1. 继电器控制电路

```
ESP8266 GPIO16 (D0)
       │
       ├───[1kΩ]───┬─── B (S8050 NPN)
       │           │
       │          GND (通过 10kΩ下拉)
       │
      GND

S8050 三极管:
  - B (基极): GPIO16 通过 1kΩ电阻
  - C (集电极): 继电器线圈负极
  - E (发射极): GND

继电器线圈:
  - 正极：5V
  - 负极：S8050 集电极
  - 续流二极管：1N4148 反向并联

继电器触点:
  - COM: 220V AC 火线输入
  - NO (常开): 连接水泵
  - NC (常闭): 空置
```

### 2. PAN3031 LoRa 电路

```
ESP8266          PAN3031
───────          ───────
GPIO14 (D5) ──── SCK
GPIO13 (D7) ──── MOSI
GPIO12 (D6) ──── MISO
GPIO15 (D8) ──── CS
GPIO2 (D4) ───── IRQ (可选)
3.3V ──────────── VCC
GND ───────────── GND
GPIO16 (D0) ───── RST (可选)

天线: 433MHz 弹簧天线 (SMA 接口)
```

### 3. OLED 显示电路

```
ESP8266          OLED 0.96"
───────          ──────────
GPIO5 (D1) ───── SDA
GPIO4 (D2) ───── SCL
3.3V ──────────── VCC
GND ───────────── GND
GPIO0 (D3) ────── RST (可选)
```

### 4. 缺水传感器电路

```
ESP8266 GPIO16 (D0)
       │
       ├───[10kΩ]─── VCC (上拉)
       │
       └─── 传感器信号
           
光电液位传感器:
  - VCC: 5V
  - GND: GND
  - OUT: GPIO16 (D0)
  - 输出：有水=高电平，缺水=低电平
```

## 从机 (STC8G1K08) 详细电路

### 1. MCU 最小系统

```
STC8G1K08 SOP16
───────────────
VCC  ──── 1 ──── GND
P3.0 ──── 2 ──── P3.1
P3.2 ──── 3 ──── P3.3
P3.4 ──── 4 ──── P3.5 (缺水传感器)
P3.6 ──── 5 ──── P3.7
P1.0 ──── 6 ──── P1.1 (ADC 液位传感器)
P1.2 ──── 7 ──── P1.3
P1.4 ──── 8 ──── P1.5
P1.6 ──── 9 ──── P1.7
P2.0 ─── 10 ──── P2.1 (LoRa SCK/MISO)
P2.2 ─── 11 ──── P2.3 (LoRa MOSI/CS)
RST ─── 12 ──── VCC
GND ─── 13 ──── VCC
P5.4 ─── 14 ──── P5.5
P4.5 ─── 15 ──── P4.6
P4.7 ─── 16 ──── VCC

注意：
- RST 通过 10kΩ上拉到 VCC
- RST 通过 10uF 电容到 GND (上电复位)
- VCC 和 GND 之间加 100nF 去耦电容
```

### 2. 液位传感器电路 (4-20mA)

```
4-20mA 液位传感器
       │
       ├── VCC (24V)
       │
       ├── GND
       │
       └── OUT (4-20mA 电流)
            │
           [250Ω] 精密电阻
            │
            ├─── P1.0 (AIN0/ADC 输入)
            │
           GND

计算:
- 4mA × 250Ω = 1.0V (0%)
- 20mA × 250Ω = 5.0V (100%)
- STC8G ADC 参考电压：5V
```

### 3. 缺水传感器电路

```
STC8G1K08 P3.4
       │
       ├───[10kΩ]─── VCC (上拉)
       │
       └─── 传感器信号

光电液位传感器:
  - VCC: 5V
  - GND: GND
  - OUT: P3.4
  - 输出：有水=高电平，缺水=低电平
```

### 4. PAN3031 LoRa 电路 (从机)

```
STC8G1K08        PAN3031
─────────        ───────
P2.0 ─────────── SCK
P2.2 ─────────── MOSI
P2.1 ─────────── MISO
P2.3 ─────────── CS
3.3V ─────────── VCC
GND ──────────── GND

注意:
- STC8G 是 5V MCU，PAN3031 是 3.3V
- 需要在信号线上加 3.3V 稳压二极管或电平转换
- 或者直接使用 3.3V LDO 给 STC8G 供电 (如果支持)
```

## 电源系统

### 主机电源

```
220V AC ────> 5V DC 电源适配器 (2A) ────┬────> ESP8266 (5V)
                                        │
                                        ├────> 继电器线圈 (5V)
                                        │
                                        ├────> OLED 显示 (3.3V LDO)
                                        │
                                        └────> PAN3031 (3.3V LDO)
```

### 从机电源

```
220V AC ────> 5V DC 电源适配器 (1A) ────┬────> STC8G1K08 (5V)
                                        │
                                        ├────> 液位传感器 (24V 单独供电)
                                        │
                                        ├────> 缺水传感器 (5V)
                                        │
                                        └────> PAN3031 (3.3V LDO)
```

## 安全注意事项

### 高压安全
1. **220V AC 部分必须隔离**:
   - 继电器触点与线圈隔离电压 ≥ 2500V
   - PCB 走线间距 ≥ 3mm
   - 加绝缘外壳

2. **防水防潮**:
   - 所有户外接线盒 IP65 防护
   - 传感器电缆使用防水接头
   - PCB 喷涂三防漆

3. **过载保护**:
   - 继电器额定电流 ≥ 水泵电流 × 2
   - 添加保险丝 (建议 5A)
   - 接地保护

### 低压安全
1. **ESD 防护**:
   - 所有 IO 口加 TVS 二极管
   - PCB 加接地铜皮

2. **电源滤波**:
   - 电源入口加共模电感
   - VCC 加 100uF + 100nF 电容
