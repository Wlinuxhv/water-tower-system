# 弹簧电极式水位传感器 - 完整设计文档

## 📋 方案概述

基于**连通器原理** + **电容检测**的水位传感器，使用弹簧作为电极，PE 管隔离水电。

```
1. 裁剪内管到指定长度 (1.5 米)

2. 标记弹簧位置:
   从底部开始:
   - 15cm: 弹簧 1 (10%)
   - 25cm: 弹簧 2 (20%)
   - 35cm: 弹簧 3 (30%)
   - 45cm: 弹簧 4 (40%)
   - 55cm: 弹簧 5 (50%)
   - 65cm: 弹簧 6 (60%)
   - 75cm: 弹簧 7 (70%)
   - 85cm: 弹簧 8 (80%)
   - 95cm: 弹簧 9 (90%)

3. 底部钻进水孔:
   在底部 2cm 处，均匀分布钻 3-4 个Φ5mm 小孔

   ┌─────────────────┐
   │                 │
   │   ○         ○   │  ← Φ5mm 进水孔
   │                 │
   │       ○         │
   │                 │
   └─────────────────┘
```

### 步骤 3: 安装弹簧电极

```
1. 缠绕弹簧:
   从底部开始，按标记位置缠绕弹簧
   
   内管外壁
      │
      ├────╔════════╗────┤ 弹簧 9
      │    ║        ║    │
      │    ╚════════╝    │
      │       │          │
      │    导线焊接点     │
      │                  │
      ├────╔════════╗────┤ 弹簧 8
      │    ║        ║    │
      │    ╚════════╝    │
      │       │          │
      │    导线焊接点     │
      
2. 焊接导线:
   - 每个弹簧焊接一根导线
   - 使用不同颜色区分
   - 焊点用热缩管绝缘

3. 固定弹簧:
   - 使用热熔胶固定弹簧两端
   - 确保弹簧不会滑动
   - 导线沿管壁引出
```

### 步骤 4: 组装外管

```
1. 内管装入外管:
   - 小心将内管放入外管中心
   - 确保弹簧在间隙中
   - 导线从顶部引出

2. 底部密封:
   - 外管底部用 ABS 圆片封闭
   - 热熔胶固定
   - 确保防水

3. 顶部固定:
   - 安装上密封盖
   - 导线从中心孔引出
   - 热熔胶密封
```

### 步骤 5: 安装 PCB

```
1. 焊接 SC09B:
   ┌─────────────────────┐
   │      PCB 板         │
   │                     │
   │   ┌───────────┐     │
   │   │  SC09B    │     │
   │   │  (焊接)   │     │
   │   └───────────┘     │
   │                     │
   │   接线端子 (焊接)    │
   │   1-9: 弹簧导线      │
   │   VCC/GND: 电源      │
   │   SCL/SDA: 信号      │
   └─────────────────────┘

2. 连接导线:
   - 弹簧 1-9 → IN1-IN9
   - VCC → 3.3V
   - GND → GND
   - SCL/SDA → 接线端子

3. 灌胶密封:
   - PCB 底部涂环氧树脂
   - 固化 24 小时
   - 确保防水
```

### 步骤 6: 连接接线盒

```
1. 安装 4 芯电缆:
   传感器 ──► 接线盒
   - VCC (红)
   - GND (黑)
   - SCL (黄)
   - SDA (绿)

2. 接线盒内部:
   ┌─────────────────────┐
   │    接线盒 (IP65)    │
   │                     │
   │  ┌───────────────┐  │
   │  │  STC8G 从机    │  │
   │  │  + LoRa 模块   │  │
   │  └───────────────┘  │
   │                     │
   │  接线端子:          │
   │  - 传感器输入       │
   │  - 电池输入         │
   └─────────────────────┘

3. 密封接线盒:
   - 电缆入口用胶密封
   - 盒盖加防水垫圈
   - 螺丝紧固
```

## 🧪 测试与校准

### 测试步骤

```
1. 气密性测试:
   ┌─────────────┐
   │   传感器    │
   │             │
   │     ▼       │
   │   ┌───┐     │
   │   │水 │     │  ← 浸入水中 10cm
   │   └───┘     │
   │             │
   └─────────────┘
   
   - 浸入水中 10cm
   - 观察 30 分钟
   - 检查是否有气泡
   - 确认无漏水

2. 电气测试:
   - 万用表测量各弹簧导通
   - 检查 I2C 通信
   - 读取 SC09B 状态

3. 水位测试:
   空塔 ──► 10% ──► 50% ──► 90% ──► 满塔
   记录各通道触发点
```

### 校准方法

```
1. 空塔校准:
   - 确保所有弹簧干燥
   - 读取 SC09B 数据
   - 应全为 0

2. 满塔校准:
   - 完全浸入水中
   - 读取 SC09B 数据
   - 应全为 1

3. 中间点校准:
   水位 10% ──► 记录通道 1
   水位 20% ──► 记录通道 1,2
   ...
   水位 90% ──► 记录所有通道

4. 软件校准:
   根据测试结果调整阈值
```

## ⚠️ 注意事项

### 1. 防水处理

```
关键密封点:
┌─────────────────┐
│  ① 底部灌胶     │ ← 最重要
│  ② 顶部密封盖   │
│  ③ 导线引出孔   │
│  ④ 接线盒入口   │
│  ⑤ PCB 灌胶     │
└─────────────────┘

所有密封点必须:
- 清洁表面
- 均匀涂胶
- 充分固化 (24 小时)
- 测试气密性
```

### 2. 弹簧安装

- 弹簧不能接触外管
- 弹簧间距均匀 (10cm)
- 导线焊接牢固
- 焊点绝缘处理

### 3. 电气安全

- VCC 使用 3.3V (不要超过 6.5V)
- I2C 加上拉电阻 (4.7kΩ)
- 使用屏蔽电缆
- 接地良好

### 4. 维护建议

- 每半年检查一次
- 清洁外管外壁 (防藻类)
- 检查密封性
- 校准水位检测
