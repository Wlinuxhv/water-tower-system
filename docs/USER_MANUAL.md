## 1. 产品概述

### 1.1 产品简介

WTMS-V2.1 水塔监控系统是一款基于 ESP8266 和 LoRa 技术的智能水位监控系统，适用于多水塔场景的自动化管理。

**核心功能**:
- 🌊 实时水位监测 (0-100%)
- 💧 自动水泵控制
- 📡 LoRa 无线通信 (1-3km)
- 📱 手机 APP 远程监控
- 🖥️ OLED 本地显示
- ⚠️ 缺水保护
- 🔧 8 路继电器控制

### 1.2 应用场景

- 农村多户水塔共享
- 小区二次供水系统
- 农业灌溉水塔
- 工业蓄水池监控

## 4. 安装指南

### 4.1 主机安装

#### 4.1.1 硬件连接

```
1. 连接 LoRa 天线
   └─→ 433MHz 弹簧天线 (SMA 接口)

2. 连接继电器模块
   └─→ 74HC595 输出 → 8 路继电器 IN1-IN8

3. 连接水泵电源
   └─→ 继电器 COM → 220V AC 火线
   └─→ 继电器 NO  → 水泵火线
   └─→ 零线直连水泵

4. 连接缺水传感器
   └─→ GPIO16 (D0) → 传感器信号
   └─→ 5V → 传感器 VCC
   └─→ GND → 传感器 GND

5. 连接电源
   └─→ 5V DC 2A 电源适配器
```

#### 4.1.2 接线图

```
220V AC 输入
    │
    ├── L ──→ 继电器 COM
    │            │
    │            ├── NO1 ──→ 水泵 1 L
    │            ├── NO2 ──→ 水泵 2 L
    │            └── ...
    │
    └── N ──┬──→ 水泵 1 N
            ├──→ 水泵 2 N
            └──→ ...

5V DC 电源
    │
    ├── VCC ──→ ESP8266 5V
    │           └─→ 继电器 VCC
    │
    └── GND ──→ ESP8266 GND
                └─→ 继电器 GND
```

### 4.2 从机安装

#### 4.2.1 传感器安装

1. **液位传感器** (4-20mA)
   - 安装位置：水塔底部
   - 连接方式：螺纹安装 (G1/2")
   - 接线：
     ```
     棕色 → 24V+
     蓝色 → GND
     黑色 → 信号 (4-20mA)
     ```

2. **缺水传感器** (光电式)
   - 安装位置：水井/水源处
   - 连接方式：螺纹或卡扣
   - 接线：
     ```
     红色 → 5V
     黑色 → GND
     黄色 → 信号 (P3.4)
     ```

### 4.3 系统配置

#### 4.3.1 设置从机 ID

通过 DIP 开关设置从机 ID (1-8):

| ID | SW1 | SW2 | SW3 |
|----|-----|-----|-----|
| 1  | ON  | OFF | OFF |
| 2  | OFF | ON  | OFF |
| 3  | ON  | ON  | OFF |
| 4  | OFF | OFF | ON  |
| 5  | ON  | OFF | ON  |
| 6  | OFF | ON  | ON  |
| 7  | ON  | ON  | ON  |
| 8  | OFF | OFF | OFF |

#### 4.3.2 配置 WiFi

1. 手机连接热点：`WaterTower_XXXX`
2. 浏览器访问：`192.168.4.1`
3. 输入 WiFi 账号密码
4. 保存重启

## 5. 操作说明

### 5.1 OLED 显示界面

#### 5.1.1 主界面

```
Mode:AUTO | Well:OK
T0:85% [PUMP]
T1:45%     
T2:92% [PUMP]
```

**显示内容**:
- 第 1 行：模式 | 井水状态
- 第 2-4 行：水塔 ID: 水位% [水泵状态]

#### 5.1.2 错误界面

```
=== ERROR ===
Code: E00101
Level: CRIT!
-------------
Well Water Low
```

### 5.2 控制模式

#### 5.2.1 自动模式 (AUTO)

系统根据水位自动控制水泵:
- 水位 < 20% → 开启水泵
- 水位 > 90% → 关闭水泵
- 井水缺水 → 紧急停止

#### 5.2.2 手动模式 (MANUAL)

通过 APP 或 Web 界面手动控制:
- 访问：`http://主机 IP`
- 点击水泵开关
- 实时状态显示

### 5.3 Web 界面操作

#### 5.3.1 访问方法

1. 确保手机/电脑与主机在同一 WiFi
2. 浏览器输入主机 IP (如 `192.168.1.100`)
3. 或访问：`http://watertower.local`

#### 5.3.2 功能菜单

- **状态监控**: 实时水位、水泵状态
- **手动控制**: 开启/关闭水泵
- **模式切换**: AUTO/MANUAL
- **历史记录**: 查看运行记录
- **系统设置**: WiFi、阈值等

## 6. 错误码手册

### 6.1 错误码格式

**显示格式**: `Exx-yyy`
- `xx`: 错误类别
- `yyy`: 具体错误编号

**示例**: `E101` = 井水缺水

### 6.2 错误码列表

#### 系统错误 (0xx)

| 错误码 | 显示 | 说明 | 级别 | 处理措施 |
|--------|------|------|------|----------|
| 0 | System OK | 系统正常 | - | - |
| 1 | SYS Init Fail | 系统初始化失败 | 🔴 严重 | 重启系统 |
| 2 | Mem Low | 内存不足 | 🟡 警告 | 重启系统 |
| 3 | WDT Reset | 看门狗复位 | 🔴 错误 | 检查程序 |
| 4 | Power Low | 电源电压低 | 🔴 严重 | 更换电源 |
| 5 | Overheat | 系统过热 | 🔴 错误 | 改善散热 |

#### 传感器错误 (1xx)

| 错误码 | 显示 | 说明 | 级别 | 处理措施 |
|--------|------|------|------|----------|
| 101 | Well Water Low | 井水缺水 | 🔴 严重 | 等待水源恢复 |
| 102 | ADC Fail | ADC 读取失败 | 🔴 错误 | 检查传感器 |
| 103 | Sensor Range | 数据超范围 | 🟡 警告 | 校准传感器 |
| 104 | Sensor Disc | 传感器断开 | 🔴 错误 | 检查接线 |

#### 通信错误 (2xx)

| 错误码 | 显示 | 说明 | 级别 | 处理措施 |
|--------|------|------|------|----------|
| 201 | LoRa Timeout | LoRa 超时 | 🔴 错误 | 检查天线 |
| 202 | LoRa CRC Err | CRC 校验失败 | 🔴 错误 | 检查干扰 |
| 203 | I2C Fail | I2C 通信失败 | 🔴 错误 | 检查 OLED |
| 204 | SPI Fail | SPI 通信失败 | 🔴 错误 | 检查 74HC595 |
| 205 | WiFi Fail | WiFi 连接失败 | 🟡 警告 | 检查路由 |
| 206 | Web Server Err | Web 服务器错误 | 🟡 警告 | 重启系统 |

#### 继电器错误 (3xx)

| 错误码 | 显示 | 说明 | 级别 | 处理措施 |
|--------|------|------|------|----------|
| 301 | Relay Stuck On | 继电器粘连 | 🔴 严重 | 更换继电器 |
| 302 | Relay Stuck Off | 继电器卡死 | 🔴 错误 | 更换继电器 |
| 303 | Relay Overload | 继电器过载 | 🔴 严重 | 减小负载 |
| 304 | 74HC595 Fail | 驱动芯片故障 | 🔴 错误 | 更换 74HC595 |

#### 网络错误 (4xx)

| 错误码 | 显示 | 说明 | 级别 | 处理措施 |
|--------|------|------|------|----------|
| 401 | No Slave Data | 无从机数据 | 🟡 警告 | 检查从机 |
| 402 | Net Timeout | 网络超时 | 🔴 错误 | 检查距离 |
| 403 | Invalid Data | 无效数据包 | 🟡 警告 | 检查干扰 |

#### 水塔错误 (5xx)

| 错误码 | 显示 | 说明 | 级别 | 处理措施 |
|--------|------|------|------|----------|
| 501 | Tower Overflow | 水塔溢出 | 🔴 严重 | 检查传感器 |
| 502 | Tower Dry | 水塔干涸 | 🟡 警告 | 检查进水 |
| 503 | Pump Fail | 水泵故障 | 🔴 严重 | 检修水泵 |
| 504 | Tower Sen Fail | 传感器故障 | 🔴 错误 | 更换传感器 |

### 6.3 错误级别说明

| 级别 | 图标 | 说明 | 系统响应 |
|------|------|------|----------|
| 正常 | ✅ | 系统正常运行 | 无 |
| 信息 | ℹ️ | 提示信息 | 记录日志 |
| 警告 | ⚠️ | 需要注意 | OLED 显示 |
| 错误 | ❌ | 功能异常 | 声光报警 |
| 严重 | 🚨 | 紧急停止 | 停机保护 |

### 6.4 错误处理流程

```
检测到错误
    │
    ▼
记录错误日志
    │
    ▼
判断错误级别
    │
    ├── 正常 → 继续运行
    │
    ├── 警告 → OLED 显示，记录日志
    │
    ├── 错误 → 报警，尝试恢复
    │
    └── 严重 → 紧急停机，等待人工干预
```

## 8. 常见问题

### Q1: 水泵不启动？

**可能原因**:
- 水位未达到启动阈值
- 继电器故障
- 水泵电源未接通

**解决方法**:
1. 检查水位是否 < 20%
2. 手动模式测试继电器
3. 检查 220V 电源

### Q2: 显示"井水缺水"？

**可能原因**:
- 水源确实缺水
- 缺水传感器故障
- 接线松动

**解决方法**:
1. 检查水源水位
2. 清洁传感器
3. 重新接线

### Q3: LoRa 通信不稳定？

**可能原因**:
- 距离过远
- 有遮挡物
- 天线未接好

**解决方法**:
1. 缩短距离 (<3km)
2. 改善天线位置
3. 检查天线连接

### Q4: WiFi 连不上？

**可能原因**:
- WiFi 密码错误
- 信号弱
- 路由器问题

**解决方法**:
1. 重新配置 WiFi
2. 靠近路由器
3. 重启路由器

### Q5: 水位显示不准？

**可能原因**:
- 传感器需要校准
- 传感器损坏
- 水中有杂质

**解决方法**:
1. 重新校准传感器
2. 更换传感器
3. 清洁水塔
